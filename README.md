# 🧪 Home Lab Workshop  
### Build Your Own Personal Mini Data-Center

A **Home Lab** is a personal computing laboratory created at home using physical or virtualized hardware.  
It allows you to **experiment, learn, prototype, and test technologies safely** in a controlled environment.

Many concepts taught in courses such as **Operating Systems, Networking, Databases, Virtualization, Cloud, and Security** are only fully understood when applied in real scenarios. A Home Lab bridges this gap by providing a **practical playground** where experimentation is encouraged and mistakes have no real-world impact.

This repository supports a **hands-on workshop** where participants will design and build their own **personal Home Lab**, evolving it into a **mini data-center for experimentation and learning**.

---

## 🚀 Why Build a Home Lab?

A Home Lab enables you to:

- 💡 Transform **theoretical knowledge into practical skills**
- 🔬 Experiment with technologies **without risking production systems**
- 🔁 Repeat experiments and iterate quickly
- 🧱 Build complex infrastructures step by step
- 🎯 Develop skills aligned with **industry practices**

It becomes your **personal innovation environment**, where you can explore technologies at your own pace and extend learning beyond formal curricula.

---

## 🧭 Workshop Overview

During this workshop, participants will learn how to:

- Plan and design a **functional Home Lab architecture**
- Select appropriate **technologies and infrastructure**
- Deploy and interconnect services
- Organize services into a coherent environment
- Create a **personal mini data-center**

By the end, each participant will have a **working Home Lab foundation** that can be continuously expanded.

---

## 🧱 What You Will Learn

Participants will:

- 🛠 **Plan, build, and evolve** a functional Home Lab  
- 🖥 Gain practical experience with **virtualization and service orchestration**  
- 🔗 **Install, configure, and interconnect** multiple services  
- 🏗 Create a **personal infrastructure platform** for experimentation  
- 📈 Develop an environment that can grow into a **personal mini data-center**

---

## 📚 Prerequisites

Participants should have basic knowledge of:

- Operating Systems (Linux recommended)
- Docker / Containerization
- Computer Networks
- Databases

---

## 💻 Required Resources

### Laptop

Participants should bring a laptop with:

| Requirement | Recommended |
|-------------|-------------|
| Docker / Docker Desktop | Installed |
| CPU | 4 cores or more |
| RAM | 8 GB minimum (16 GB preferred) |
| Storage | At least 80 GB free |

---

### Optional Hardware

Participants who want to build their Home Lab on dedicated hardware may bring:

#### 🖥 Extra Machine
- A spare computer with **minimal Ubuntu Server installation**
- **SSH server installed**

#### 🍓 Raspberry Pi
- **Raspberry Pi 5 (16GB recommended)**
- Fresh installation will be performed during the workshop

---

## 🏗 Example Home Lab Stack

A typical Home Lab environment may include:

- **Docker / Docker Compose**
- **Reverse Proxy / Gateway**
- **Service Dashboard**
- **DNS / Network utilities**
- **Databases**
- **Monitoring tools**
- **Development services**

Over time, this can evolve into a **personal cloud platform**.
